# Hi, I'm Alex 👋🏻

I'm a **Full-Stack Engineer** who enjoys building clean, performant, and well-tested web applications end-to-end — from React UIs to Express APIs with PostgreSQL. I'm particularly interested in integrating AI into real-world products and constantly exploring what's new in the TypeScript ecosystem.

---

## 🧑‍💻 About Me

- 🔭 Currently building a full-stack AI-powered expense tracker: React 19 + Bun/Express backend + PostgreSQL, with OpenAI GPT-4o for email-based transaction ingestion.
- 🤖 Interested in practical AI integration — structured extraction with the Vercel AI SDK, Zod schemas, and confidence-based staging flows.
- 🎨 Experienced crafting polished UIs with **shadcn/ui**, **Tailwind CSS v4**, and design-system thinking.
- ⚡ Care deeply about performance, type-safety, code quality, and developer experience.
- 🧪 Write unit tests (Vitest + RTL) and end-to-end tests (Playwright) as a first-class concern.
- 🤝 Open to collaborating on interesting open-source projects.
- 📫 Reach me at [alex_bb23@yahoo.co.uk](mailto:alex_bb23@yahoo.co.uk)
